# PyBlog
Python &amp; Html5(CSS,JS) => Blog
## 开始使用
- 确保你的服务器安装了Python3.8
- 确保你的服务器安装了BT
- 安装`yum install -y unzip zip`
<br>
<br>
下载文件[Releases](https://github.com/underthestars-zhy/PyBlog/releases)
<br>
运行

```python3 main.py```
<br>
然后请安装**main.py**的提示进行操作
<br>
<br>
删除数据请运行
<br>
```python3 clean.py```
<br>
